The post-tag is added automatically if that is found a content when saving post. Moreover, some similar words can be set to a post-tag.

== Description ==

It becomes unnecessary for a author to add a few post-tag manually by using this plug-in. It is because it confirms whether a post-tag is contained when a post is published. The found post-tag is automatically added by this plug-in.

= Some features: =

* This plug-in can register some similar words of a post-tag.
* When a post is published, it is confirmed whether all post-tag and some similar words are included in a title and a content. And the post-tag will be added when they are found.
* All the posts can be confirmed at once.
